搭建博客

2021-03-13
技术
约 496 字 预计阅读 1 分钟

Photo by Glenn Carstens-Peters on Unsplash

最近心血来潮，折腾下自己的博客网站。

对比了 Jekyll/Hugo/Hexo 三种方案，结合三者在GitHub 上 受欢迎程度以及背后的技术语言，最终选用的技术是 Github Pages + Hugo。

在这两天的使用中，印象最深的还是 Hugo 的修改实时显示，非常方面，刷新页面都成了多此一举。

评论模块。可选的非常多，对比了 Disqus、Valine、Utterances、Gitalk、Gitment。 最终选择 Utterances。Even 主题已经支持 Utterances，参照教程，操作前4步就行，非常简单。

— 2021-04-05 —

个人域名 forgeyao.tech 上线。腾讯云上注册的，主要是便宜，10 年只要 ¥198，还有免费证书支持。

注意：配置域名 DNS cname 到 forgeyao.github.io，然后在 GitHub 上设置个性域名后。
还要在 config.toml 里把 baseURL 换成新域名，不然 sitemap.xml 会出现跨域无法访问。因为其他页面都是正常的，所以还有点难发现。

网站提交到 Google Search Console 收录。（很难理解没自动收录，虽然是小网站）

下面是还需要进一步完善。

  1. SEO
  2. Google Analytics 使用
  3. 持续写作。熟悉 Markdown 语法，适应 Markdown 写作
